Conceitos de Programação

O curso de Tecnologia em Sistemas para Internet do IFTM aborda, ao longo de sua grade curricular, os principais fundamentos da programação moderna. A seguir, são apresentados os conceitos centrais trabalhados nas disciplinas do curso.

Lógica de Programação

A Lógica de Programação é a base de todo o aprendizado em desenvolvimento de software. Ela consiste em desenvolver o raciocínio necessário para resolver problemas por meio de sequências de instruções claras e ordenadas, chamadas de algoritmos.

Um algoritmo é uma sequência finita de passos que leva à solução de um problema. Antes de escrever qualquer código em uma linguagem de programação, o programador deve ser capaz de descrever a solução de forma lógica e estruturada.

Entre os principais conceitos estudados estão as estruturas de controle, que permitem tomar decisões durante a execução de um programa. A estrutura condicional se/senão (if/else) verifica uma condição e executa blocos de código diferentes conforme o resultado. Já as estruturas de repetição, como o enquanto (while) e o para (for), permitem executar um trecho de código várias vezes, enquanto uma condição for verdadeira ou por um número determinado de iterações.

Também fazem parte da Lógica de Programação o uso de variáveis, que são espaços na memória do computador destinados a armazenar valores; os tipos de dados, como inteiro, real, caractere e booleano; e as operações aritméticas e lógicas, que permitem realizar cálculos e comparações entre valores.

O domínio da lógica de programação é indispensável para o aprendizado de qualquer linguagem, pois os conceitos se aplicam universalmente, seja em Python, Java, C, JavaScript ou qualquer outra tecnologia.

Orientação a Objetos

A Programação Orientada a Objetos (POO) é um paradigma de desenvolvimento de software que organiza o código em torno de objetos, que representam entidades do mundo real ou conceitos abstratos do sistema. Esse modelo facilita a organização, a reutilização e a manutenção do código.

O conceito fundamental da POO é a classe, que funciona como um molde ou modelo para a criação de objetos. Uma classe define os atributos (características) e os métodos (comportamentos) que seus objetos terão. Por exemplo, uma classe "Aluno" pode ter os atributos nome, matrícula e curso, e os métodos realizarMatrícula() e consultarNotas().

Um objeto é uma instância de uma classe, ou seja, uma entidade criada a partir desse molde com valores específicos para seus atributos. Vários objetos podem ser criados a partir de uma mesma classe, cada um com seus próprios dados.

A POO é sustentada por quatro pilares principais. O encapsulamento consiste em proteger os dados internos de um objeto, expondo apenas o necessário por meio de métodos públicos. A herança permite que uma classe filha herde atributos e métodos de uma classe pai, promovendo o reaproveitamento de código. O polimorfismo permite que objetos de classes diferentes respondam de maneira distinta a um mesmo método. Por fim, a abstração consiste em representar apenas as características essenciais de um objeto, ocultando detalhes de implementação desnecessários.

No IFTM, a Orientação a Objetos é trabalhada principalmente nas disciplinas de Java e Python, linguagens amplamente utilizadas no mercado e que implementam esse paradigma de forma robusta.

Desenvolvimento Web

O Desenvolvimento Web é uma das áreas centrais do curso de Sistemas para Internet e abrange o conjunto de tecnologias e práticas utilizadas para a criação de aplicações e páginas acessíveis por meio de navegadores. Ele é dividido em duas frentes principais: o desenvolvimento front-end e o desenvolvimento back-end.

O front-end é responsável pela parte visual e interativa de um site, ou seja, tudo aquilo que o usuário vê e com que interage diretamente. As principais tecnologias envolvidas são o HTML (HyperText Markup Language), que estrutura o conteúdo da página; o CSS (Cascading Style Sheets), que define a aparência visual, como cores, fontes e layout; e o JavaScript, que adiciona interatividade e dinamismo às páginas.

O back-end compreende a lógica de negócios e o processamento de dados que ocorrem no servidor, de forma invisível ao usuário final. É no back-end que são realizadas operações como autenticação de usuários, consultas a banco de dados, envio de e-mails e processamento de formulários. Linguagens como PHP, Python e Java são frequentemente utilizadas nessa camada.

Outro conceito importante é o de banco de dados, que armazena as informações manipuladas pela aplicação. A comunicação entre o back-end e o banco de dados é feita por meio da linguagem SQL (Structured Query Language), que permite criar, consultar, atualizar e remover registros.

Atualmente, o desenvolvimento web também envolve o uso de frameworks, que são conjuntos de ferramentas e bibliotecas que agilizam o processo de desenvolvimento. No front-end, destacam-se frameworks como React e Vue.js. No back-end, são amplamente utilizados o Django (Python), o Laravel (PHP) e o Spring (Java).

O IFTM prepara seus estudantes para atuar em todas essas camadas, oferecendo disciplinas que cobrem desde os fundamentos do HTML e CSS até o desenvolvimento de aplicações web completas e responsivas, prontas para o mercado de trabalho.
